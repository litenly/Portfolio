Основы вёрстки

В этом блоке я изучила язык гипертекстовой разметки HTML и каскадные таблицы стилей CSS. Если HTML отвечает за структуру и содержание сайта, то CSS определяет, как этот контент будет выглядеть: цвета, шрифты, отступы, расположение элементов и многое другое.

Пройденный материал

синтаксис HTML, вставка контента

HTML — это основа веб-страниц, позволяющая структурировать и оформлять контент, а также встраивать мультимедийные элементы. Основные теги для работы с текстом p — абзац. h1–h6 — заголовки разных уровней. span — встроенный элемент для выделения части текста. div— блок для группировки контента. br — перенос строки. hr — горизонтальная линия. Вставка мультимедийного и встраиваемого контента HTML позволяет добавлять на страницу изображения, видео, аудио, документы и другие объекты с помощью тегов: img — изображение. video — видео. audio — аудио. iframe, embed, object — встраивание внешних ресурсов и плагинов. Форматирование и структурирование текста Для выделения текста используются теги: strong (жирный), em (курсив), u (подчёркивание), /s (перечёркивание), mark (маркер).Для списков применяются теги ul, ol, li. Важные моменты: Правильная структура (заголовки, абзацы, списки) улучшает читаемость и доступность страницы. Всегда закрывайте теги ( < > - первый тег, < / > - закрывающий тег) и используйте отступы для удобства чтения кода.

Синтаксис CSS, стилизация контента

CSS — ключевой инструмент для создания красивых, функциональных и современных веб-страниц.Синтаксис CSS Селектор — указывает, к каким HTML-элементам применять стиль. Объявление — пара «свойство: значение», задающая конкретный визуальный эффект. Пример: css Копировать p { color: blue; font-size: 16px; } Основные возможности стилизации Текст: font-family, font-size, font-weight, text-align, line-height и др. Цвет: color, background-color, border-color, тени (box-shadow, text-shadow). Цвета можно задавать именами, HEX, RGB, HSL. Блочная модель (Box Model) Каждый элемент состоит из: Содержимого (content). Внутренних отступов (padding). Границы (border). Внешних отступов (margin). Позиционирование и макеты Позиционирование: static, relative, absolute, fixed, sticky. Современные методы: Flexbox — для гибких строк и колонок. Grid — для сложных сеточных макетов. Адаптивная верстка Используются медиа-запросы (media queries), чтобы стили менялись в зависимости от размера экрана устройства. Дополнительные возможности Анимации и переходы. Градиенты, тени, трансформации. Псевдоклассы (:hover, :focus) и псевдоэлементы (::before, ::after).

Макетирование, способы расположения контента

Современное макетирование с помощью CSS позволяет создавать гибкие, адаптивные и красивые интерфейсы для любых устройств.Способ Назначение Особенности Нормальный поток - Стандартное расположение элементов по порядку в HTML Блочные элементы идут друг под другом, строчные — в строке Flexbox - Одномерные макеты (строка или столбец) Удобен для навигационных панелей, списков, карточек, выравнивания и распределения пространства CSS Grid - Двумерные макеты (сетки: строки и столбцы) Идеален для сложных структур: страницы с боковой панелью, галереи, формы, адаптивные сетки Floats - Обтекание элементов (например, изображения в тексте) Используется реже, в основном для специфических задач Позиционирование (position) - Точное размещение отдельных элементов Виды: static, relative, absolute, fixed, sticky Макет таблицы (display: table) - Табличное расположение элементов Применяется редко, в основном для табличных данных Многоколоночный макет - Расположение контента в несколько колонок Как в газетах, для длинных текстов
