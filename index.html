Программы, в которых я работала

figma

Figma — это онлайн-платформа для проектирования интерфейсов, совместной работы и создания макетов сайтов, мобильных приложений, презентаций и других дизайнерских проектов. Она позволяет нескольким пользователям работать над одним файлом в реальном времени, что упрощает коммуникацию между дизайнерами, разработчиками, менеджерами и другими участниками проекта.

Adobe Photoshop

Adobe Photoshop — профессиональный растровый графический редактор от компании Adobe Inc. Программа предназначена для создания, редактирования и обработки цифровых изображений: работы с фотографиями, графического дизайна и цифровой живописи.

Visual Studio Code

Visual Studio Code (VS Code) — бесплатный кросс‑платформенный редактор кода от Microsoft. Подходит для разработки на разных языках программирования и создания кросс‑платформенных приложений. Работает на Windows, macOS и Linux; есть веб‑версия (vscode.dev).
